# 🏥 MediTrack - Multi-Portal Healthcare Ecosystem
**Industrial Training Project | Powered by DecodeLabs (Batch 2026)**

MediTrack is a high-performance, full-stack medical management system. It has evolved from a simple UI into a secure, stateful platform featuring a **"Digital Vault"** powered by PostgreSQL and a **"Gatekeeper"** security layer.

---

## 🚀 The Three-Tier Evolution

* **Skin (Frontend):** Interactive multi-portal UI with **Energetic Glassmorphism** and a **Dynamic Dark Mode**.
* **Brain (API):** A secure RESTful backend built with Node.js and Express.
* **Vault (Persistence):** 100% data longevity using a professional PostgreSQL relational schema.

---

## 🛠️ Technical Stack

* **Frontend:** HTML5, CSS3 (Advanced Grid/Flexbox), JavaScript (ES6+).
* **Backend:** Node.js, Express.js.
* **Database:** PostgreSQL (Relational Geometry).
* **Security:** Parameterized Queries ($1, $2...), Bcrypt password hashing, and Role-Based Access Control (RBAC).
* **DevOps/Tools:** Git, GitHub, DBeaver, `pg` (Postgres client).

---

## 💎 Engineering Highlights

### 1. Persistence & Data Integrity
* **Relational Schema:** Designed tables with strict Foreign Key constraints for absolute reliability.
* **Performance:** Optimized queries to ensure high-speed data retrieval for schedules and rosters.

### 2. Multi-Portal Authentication
* **Role-Based Access:** Separate secure portals for **Admin, Doctor, and Patient**.
* **Doctor Verification:** A dedicated workflow for Admin to verify medical licenses.

### 3. Modern UX/UI Excellence
* **Energetic Glassmorphism:** Premium visual style using frosted-glass effects.
* **Full Responsiveness:** Fluid layouts optimized for mobile, tablet, and desktop screens.
